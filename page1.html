$6,500 - $3,600 = 0.05x

$2,900 = 0.05x

Dividing both sides by 0.05:

x = $58,000

Question 48: Incorrect

A plane loaded with cargo travels 2000 miles at an average speed of 400 Miles per hour. On the return trip on the same route it travels at an of 480 average speed miles per hour. How much quicker was the return trip?

Explanation

Using the formula:

Time = distance / speed


On the outbound trip, the distance traveled is 2000 miles and the average speed is 400 mph. So the time taken is:

Time = 2000 / 400

Time = 5 hours


On the return trip, the distance traveled is also 2000 miles, but the average speed is 480 mph. So the time taken is:

Time = 2000 / 480

Time = 4.1667 hours (rounded to 4 decimal places)


The difference in time between the two trips is:

5 - 4.1667 = 0.8333 hours (rounded to 4 decimal places)


Therefore, the return trip was 0.8333 hours, or approximately 50 minutes (since there are 60 minutes in an hour), quicker than the outbound trip.
